## java
```
1. 通过[maven](./java/pom.xml)引入bitcoinj-core.jar包，建议版本0.14.7
2. 首先生成地址和私钥，私钥需要自己保存，不可丢失，所有请求需要地址请求，签名需要私钥
3. metadata和data的内容都必须转成16进制，如果有中文需要utf-8转码，在转16进制
```
